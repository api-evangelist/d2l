# D2L Brightspace (d2l)

D2L Brightspace is a leading cloud-based learning management system (LMS) used by educational institutions and organizations worldwide to deliver, manage, and personalize learning experiences. The Brightspace Learning Framework REST API provides comprehensive programmatic access to courses, users, enrollments, grades, content, quizzes, discussions, and learning outcomes. Developers can extend and integrate Brightspace using OAuth 2.0 authentication, with SDKs available for JavaScript, Java, and .NET.

The APIs.json for this provider is at: https://raw.githubusercontent.com/api-evangelist/d2l/refs/heads/main/apis.yml

Manage this API using Naftiko: https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=d2l-api-evangelist&utm_content=repo

## APIs

| Name | Description |
|------|-------------|
| Brightspace Learning Platform (LP) API | Core API for organizational units, user accounts, enrollments, roles, and permissions. |
| Brightspace Learning Environment (LE) API | API for course content, grades, quizzes, discussions, rubrics, and intelligent agents. |
| Brightspace ePortfolio API | API for ePortfolio objects, artifacts, reflections, collections, and presentations. |
| Brightspace Data Hub (BDS) API | Bulk data export API for analytics, reporting, and data warehousing. |
| Brightspace IPSIS (SIS Integration) API | SIS integration API for automated provisioning of courses, users, and enrollments. |
